# ADR-0005 GitOps Platform

**Status:** Proposed

## Context

Application deployment should be declarative and version controlled.

## Decision

Current preference: FluxCD.

Decision will be revisited after Kubernetes deployment.

## Alternatives Considered

### FluxCD

Advantages

- Lightweight
- Git-native
- Excellent ARM compatibility

### ArgoCD

Advantages

- Rich web UI
- Mature ecosystem

Disadvantages

- Larger footprint

## Consequences

Decision pending validation.
