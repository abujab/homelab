# ADR-0004 Persistent Storage

**Status:** Proposed

## Context

Containers should survive node failures and restarts.

Persistent storage is therefore required.

## Decision

Current preference: Longhorn.

Final decision deferred until Kubernetes deployment.

## Alternatives Considered

- Local Path Provisioner
- NFS
- Longhorn
- OpenEBS
- Ceph

## Consequences

Decision pending.

Prototype before acceptance.
